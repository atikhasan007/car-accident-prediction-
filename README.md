A **Streamlit web application** to predict the **severity of road accidents** using a trained **Sklearn Pipeline**.  
The app allows users to input accident-related details and predicts whether the injury will be **Slight, Serious, or Fatal**.

---

## Features

- Predicts accident severity: **Slight, Serious, Fatal**
- Interactive **Streamlit interface** with sidebar for user inputs
- Handles multiple accident-related features including driver, vehicle, road, and casualty information
- Uses a pre-trained **Scikit-learn Pipeline model**
- Displays an image banner for better UI

---

## Prerequisites

- Python 3.8+
- Streamlit
- Pandas, NumPy
- PIL (Pillow)
- Scikit-learn
- Pickle
